Fundamentos de CSS (Cascading stylesheet)

Como diseñadores web o desarroladores tenemos a la mano el mejor lenguaje para dar estilo a nuestros sitios en HTML, la importancia de CSS en el diseño web actual radica en que nos permite separar el contenido de las páginas web de la forma de presentación.
CSS (Cascading stylesheet) o en español hojas de estilo en cascada fue desarrollado por el W3C en 1996, se denomina hojas de estilo en cascada por que las caracteristicas se aplican de arriba hacia abajo mediante reglas que poseen un esquema prioritario.
Segun el sitio web https://elrincondedanigarcia.com/css-fundamentos/ El objetivo de CSS es dar estilo a un documento HTML indicando a los navegadores como deben mostrar o que comportamiento visual tiene que tener cada uno de los elementos que compone nuestros documentos HTML.
Existen tres formas de aplicar estilos a documentos HTML:

  • Mediante el atributo style:

    El atributo style pertenece a CSS pero se inserta dentro de las etiquetas HTML directamente.

  • Mediante etiqueta style

    • Dentro del documento de HTML, usando un bloque style, generalmente cuando se usa este sistema se ubica en el head de la página.

  • Mediante un archivo CSS externo

    • Esta es la forma mas recomendada para definir los estilos ya que ofrece una gran versatilidad al momento de dar estilos por ejemplo de manera masiva, ofrece una manera mas practica de administrar los estilos aplicados al sitio web solo es necesario definir un archivo de texto con la extensión .CSS
    Para referenciar el archivo .CSS en cada archivo HTML solo es necesario referenciarlo dentro del encabezado.

Sistema de prioridades

CSS aplicara un sistema de prioridad en el cual va desde lo mas especifico a lo mas generico en cascada ya que en caso de existir reglas en el mismo nivel se ejecutara o aplicara siempre la ultima.
