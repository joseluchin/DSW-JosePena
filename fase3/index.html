Proposito del sitio

Como integrante del equipo de diseño de HtmlDesign presento este sitio el cual tiene como propósito mostrar a nuestro cliente uno de los objetos virtuales para aprendizaje virtual acerca los temas de desarrollo de sitios Web, en este caso fundamentos de CSS y frameworks CSS . de esta forma buscamos fortalecer mediante herramientas de este tipo los recursos disponibles para que los estudiantes tengan una mejor experiencia durante el desarrollo de las actividades.

Fundamentos de CSS

Según Celaya Luna(2014) CSS son las siglas de (Cascading Style Sheets) "Hojas de estilo en cascada" y parten de un concepto simple pero muy potente, básicamente se trata de aplicar a documentos principalmente HTML estilos por ejemplo (Colores, formas , márgenes etc..)preferiblemente de forma masiva.

Frameworks CSS

Según Wikipedia (2020) un framework de CSS en una biblioteca de estilos genéricos que puede ser usada para la implementación de sitios web, dichas colecciones puedes ser usadas para dar un toque personalizado a una página partiendo de estilo ya definido por otros desarrolladores, los frameworks pueden hacer que el desarrollo de sitios web sea mucho más fácil y rápido ya que ofrecen definiciones de estilos personalizadas que ahorran miles de líneas de código.
