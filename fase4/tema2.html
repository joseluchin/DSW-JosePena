Frameworks en CSS

¿Que es un Framework en CSS?

Según Wikipedia (2020) un framework de CSS en una biblioteca de estilos genéricos que puede ser usada para la implementación de sitios web, dichas colecciones puedes ser usadas para dar un toque personalizado a una página partiendo de estilo ya definido por otros desarrolladores, los frameworks pueden hacer que el desarrollo de sitios web sea mucho más fácil y rápido ya que ofrecen definiciones de estilos personalizadas que ahorran miles de líneas de código. En otras palabras, podemos definir a un Framework como una infraestructura digital dando un marco o bosquejo para la unificación de todos los elementos y componentes de un proyecto de desarrollo.

Los frameworks se clasifican en:

  • Frameworks multipropósito

    Proporcionan un conjunto de utilidades variadas útiles para distintos aspectos del desarrollo de un sitio Web.

    Los frameworsks multipropósito son los mas usados por desarrolladores front end ya que ofrecen gran versatilidad, agilidad colaboración y facilidad, pero es importante entender que el uso de frameworks no quiere decir que evitemos entender y aprender CSS, los frameworks fueron desarrollados por otros programadores desde puntos de vista de un diseño estándar esto quiere decir que al usarlos tendremos que visualizar su adaptabilidad a nuestro proyecto, por esta y otras razones es importante que antes de elegirlos tengamos una visión clara del objetivo de acuerdo a los requerimientos de nuestros clientes.

    Algunos ejemplo de frameworks multiproposito:

    • Bootstrap
    • Foundation
    • Bulma

  • Propósito especifico

    Estan orientados a proporcionar herramientas para un proposito determinado.

    Este tipo de frameworks son usados por los desarrolladores para el desarrollo de herramientas para un propósito determinado por ejemplo hay frameworks creados para crear las famosas “Breadcrumbs” o miga de pan que nos ayuda a visualizar la posición en la que estamos dentro de la página Web, también existen para crear rejillas, menús desplegables etc.
